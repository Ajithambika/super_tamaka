Send Anywhere Android SDK

Setup

Send Anywhere Android SDK is available via both jcenter() and mavenCentral(). Just add the following line to your gradle dependency:

compile ('com.estmob.android:sendanywhere:6.6.10@aar') {
    transitive = true
}
